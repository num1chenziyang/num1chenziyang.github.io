### 六、🤖 代码游乐场
````markdown
```python
# 斐波那契生成器（点击▶️运行）
def fibonacci():
    a, b = 0, 1
    while True:
        yield a
        a, b = b, a + b

print([next(fib) for fib in [fibonacci()] for _ in range(10)])
```
````
